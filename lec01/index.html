🍬 Sugar Stereo Explorer — 糖立体化学エクスプローラー

学習目標: D/L 配置、α/β アノマー、エピマー (Glc/Man/Gal) の違いを構造で識別できる。フィッシャー投影と Haworth 式を相互変換できる。

D-Glc vs D-GalC4 の OH 立体C4 エピマー

D-Glc vs L-Glcすべての不斉炭素鏡像異性体 (エナンチオマー)

α / β アノマー

単糖が環化すると C1 (アルドース) や C2 (ケトース) のカルボニル炭素が新たな不斉中心 (アノマー炭素) になり、2 つの立体配置 (α/β) が生じる。水溶液中ではα ↔ β が相互変換 (変旋光)、平衡比率は β-D-Glc 約 64% : α-D-Glc 約 36% (より安定なβが多い、すべてエクアトリアル配置)。

🔗 Glycosidic Bond Builder — グリコシド結合ビルダー

学習目標: 二糖の構造を構成単糖と結合タイプ (α/β、1→4 など) から記述できる。還元末端と非還元末端を判定できる。

二糖まとめ

二糖構成単糖結合還元性由来・特徴
マルトースGlc + Glcα1→4あり麦芽、デンプン分解物
セロビオースGlc + Glcβ1→4ありセルロース分解物
ラクトースGal + Glcβ1→4あり乳汁 (〜7%)、不耐症あり
スクロースGlc + Fruα1↔β2なし砂糖、両アノマー炭素が結合
トレハロースGlc + Glcα1↔α1なし昆虫・菌、保水性 (食品・医薬)
イソマルトースGlc + Glcα1→6ありアミロペクチン分枝部

還元糖と非還元糖

還元糖 = アノマー炭素 (C1 など) がグリコシド結合に使われていない → アルデヒド型と環状型の平衡 → フェーリング反応・ベネジクト反応で陽性。
非還元糖 = 両方のアノマー炭素が結合に使われている (スクロース・トレハロース) → 環状構造のみ → 還元性を示さない。

ラクトース不耐症 (臨床)

ラクターゼ (β-D-ガラクトシダーゼ) 活性低下 → ラクトースが小腸で分解されず結腸へ → 腸内細菌による発酵 → ガス・有機酸発生 → 腹部膨満・下痢。アジア人で頻度高 (~80-90% で活性低下)。

🌾 Polysaccharide Navigator — 多糖構造ナビゲーター

学習目標: 主要な多糖の構成単糖、結合タイプ、分子構造 (直鎖/分枝)、機能、消化酵素を対応づけられる。

貯蔵多糖 vs 構造多糖

カテゴリ例特徴
貯蔵デンプン (アミロース + アミロペクチン)、グリコーゲンα 結合 → ヘリックス構造 → 消化酵素でアクセス可
構造セルロース、キチン、ペプチドグリカンβ 結合 → 直鎖が水素結合で繊維化 → 強度高・消化困難
細胞外マトリックスヒアルロン酸、コンドロイチン硫酸、ヘパラン硫酸GAG = [アミノ糖 + ウロン酸] 反復、粘性・弾力性

ヨウ素デンプン反応

アミロース (直鎖、長いらせん) → 濃青色。アミロペクチン (分枝、短いらせん) → 赤紫色。グリコーゲン → 赤褐色。らせん内部に I₃⁻ が取り込まれて発色 (らせんが長いほど短波長吸収 → 青)。

消化のおさらい

唾液・膵液アミラーゼ = α(1→4) を内部から加水分解 → マルトース・限界デキストリン。
α-グルコシダーゼ + 脱分枝酵素 = 残存 α(1→4) と α(1→6) を分解 → グルコース → 小腸吸収。
セルロース = ヒトはセルラーゼを持たない → 食物繊維として通過。反芻動物・シロアリは共生細菌のセルラーゼで分解。

🧮 Methylation Analysis Calculator — メチル化分析計算

学習目標: アミロペクチンの分枝率を、メチル化分析の生成物分布から定量的に求められる (S44-46 演習問題)。

原理

アミロペクチンをメチル化試薬ですべての遊離 -OH を -OCH₃ に置換 → 希酸加水分解で単糖まで分解。
各 Glc 残基の位置によって、加水分解後に得られるメチル化グルコースの種類が変わる:

残基の位置加水分解後の生成物意味
非還元末端2,3,4,6-テトラ-O-Me-Glc4 ヵ所メチル化 (1 位はグリコシド側)
直鎖内部 (α1→4)2,3,6-トリ-O-Me-Glc3 ヵ所メチル化
分枝点 (α1→4 + α1→6)2,3-ジ-O-Me-Glc2 ヵ所のみ — 1, 4, 6 位はグリコシド側
還元末端1,2,3,6-テトラ-O-Me-Glc4 ヵ所 (4 位はグリコシド)、量は微量

分枝率 = (2,3-ジ-O-Me-Glc のモル数) / (全 Glc のモル数) × 100 (%)

計算の道筋 (デフォルト値の場合)

Glc の分子量 = 180 (C₆H₁₂O₆)。試料中の全 Glc モル数 ≈ 258 / 162 = 1.59 mmol (脱水縮合で 1 残基あたり -18 → 162 g/mol)。
2,3-ジ-O-Me-Glc の分子量 = 180 + 15 × 2 − 2 = 208 (-H × 2 を -CH₃ × 2 に置換)。生成量から分枝モル数 ≈ 12.4 / 208 = 0.0596 mmol。
分枝率 = 0.0596 / 1.59 × 100 ≈ 3.74 % (= 約 27 残基ごとに 1 分枝、アミロペクチンの典型値 24-30 と整合)。

💊 Antibiotic Target — 細胞壁ペプチドグリカンと抗生物質

学習目標: ペプチドグリカンの構造と、ペニシリン・バンコマイシン等の作用機序を結びつけて説明できる。グラム陽性・陰性菌の細胞壁構造の違いを理解する。

ペプチドグリカン構造

細菌の細胞壁主成分。N-アセチルグルコサミン (GlcNAc) + N-アセチルムラミン酸 (MurNAc) が β(1→4) 結合で交互に並ぶ糖鎖が、テトラペプチド側鎖で架橋された網目構造。
グラム陽性菌では ペンタグリシン橋が架橋を強化、グラム陰性菌では薄い層 + 外膜 (LPS) を持つ。

機序まとめ

薬剤分子標的機序有効菌
ペニシリンPBP (ペニシリン結合タンパク = トランスペプチダーゼ)架橋形成のトランスペプチダーゼを共有結合で阻害 → 細胞壁脆弱 → 溶菌主に グラム陽性、一部 陰性
バンコマイシンD-Ala-D-Ala 末端架橋前駆体に直接結合 → トランスペプチダーゼがアクセス不可 → 細胞壁形成阻害グラム陽性 (MRSA に有効)、陰性は外膜で通過困難
リゾチームGlcNAc-MurNAc β(1→4) 結合糖鎖を直接加水分解 (生体内防御酵素、涙・唾液)主に グラム陽性

MRSA = Methicillin-Resistant Staphylococcus aureus

通常の PBP (PBP1-3) の代わりに、ペニシリン耐性の PBP2a (mecA 遺伝子由来) を発現 → β-ラクタム系すべて (メチシリン、オキサシリン等) に耐性。バンコマイシンは D-Ala-D-Ala に直接結合するので有効 (ただし VRSA も近年出現)。
