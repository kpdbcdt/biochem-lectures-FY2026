⚖️ 電荷シミュレーター — pH を変えると電荷はどうなる?

学習目標: ヘンダーソン・ハッセルバルヒの式を使い、任意の pH における各解離基のイオン化状態と正味電荷を推定できる (スライド S29-30 の計算をインタラクティブに再現)。

ヘンダーソン・ハッセルバルヒの式

pH = pKa + log([A⁻]/[HA])
酸性基 (-COOH): pH ≫ pKa で 脱プロトン化 (-COO⁻、負電荷)。
塩基性基 (-NH₃⁺): pH ≪ pKa で プロトン化 (-NH₃⁺、正電荷)。塩基性側鎖は pH = pKa + log([B]/[BH⁺]) で考える (符号に注意)。
pH = pKa のとき各基はちょうど半分解離 ([HA]:[A⁻] = 1:1)。

📈 滴定曲線 & 等電点 (pI)

学習目標: アミノ酸の正味電荷-pH 曲線から緩衝領域 (pKa) と等電点 (pI) を読み取り、pI を「正味電荷 0 を挟む 2 つの pKa の平均」として計算できる。

等電点 (pI) の求め方

pI = アミノ酸が電気的に中性 (正味電荷 0) になる pH。このとき溶解度は最小。
・解離基 2 つ (中性アミノ酸): pI = (pK₁ + pK₂)/2
・酸性アミノ酸 (Asp/Glu): pI = 小さい 2 つの pKa の平均 (pK₁ と pK_R)
・塩基性アミノ酸 (His/Lys/Arg): pI = 大きい 2 つの pKa の平均 (pK_R と pK₂)
→ コツ: 正味電荷 0 の形を「挟む」2 つの pKa を平均する。
